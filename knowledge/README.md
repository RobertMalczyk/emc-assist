# Knowledge folder

## `seed/`

Contains pre-built knowledge bases:

- a general EMC / LTspice knowledge base,
- a PCB parasitics knowledge base,
- sources,
- rules,
- XLSX sheets for manual review.

## `raw_sources/`

Drop downloaded public source materials here. Recommended naming:

```text
<SOURCE_ID>__<short_slug>.<ext>
```

## `processed/`

The application may store summaries, chunks, and the local index here.

## Legal note

Public access to PDF / HTML does not imply the right to full redistribution. In the product, use metadata, links, short summaries, and our own rules.
